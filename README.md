# 音楽アプリ グレーアウト対策版

iPhoneのファイル選択でMP3が灰色になって選べない場合に備えて、inputのaccept指定を外した版です。

## 修正点

以前の版：
```html
<input type="file" accept=".mp3,.m4a,.aac,.wav,..." multiple>
```

今回の版：
```html
<input type="file" multiple>
```

ファイル選択時の制限を外し、取り込み後にアプリ側で以下の形式だけ保存します。

- MP3
- M4A
- AAC
- WAV

## 使い方

1. iPhoneの「ファイル」アプリに音楽フォルダを作る
2. MP3 / M4A / AAC / WAV を保存する
3. アプリで「ファイルから取り込む」を押す
4. 曲をまとめて選択する

## 注意

まだ灰色で選べない場合は、そのファイルがiCloud上にあり端末へ未ダウンロードの可能性があります。
ファイルアプリで一度そのMP3をタップしてダウンロードするか、「このiPhone内」に移動してから試してください。

音楽ファイルはIndexedDBに保存されます。
Netlifyなどのサーバーには音楽ファイルを保存しません。


## 音声運転モード
停車中に「音声運転モードを開始」を押した後、「ヘイこうだい、曲名を流して」と話します。
対応操作：曲名指定、次の曲、前の曲、停止、再生、シャッフル。
画面ロック中・バックグラウンド・ブラウザ非対応環境では動作しない場合があります。
